# AI Writing Pattern Detection Guide

> Adapted from Wikipedia's AI Cleanup project (24 patterns).
> Brand voice (brand-voice.md) ALWAYS wins on conflicts.

---

## Content Patterns

### 1. Sycophantic Openers
**Detect:** "Great question", "That's a really interesting point", "Absolutely"
**Fix:** Delete the opener. Start with substance.

### 2. Unearned Conclusions
**Detect:** "In conclusion", "Ultimately", "At the end of the day"
**Fix:** Let the last point land on its own. No summary wrapper.

### 3. Significance Inflation
**Detect:** "It's important to note", "Significantly", "It's worth mentioning", "Notably"
**Fix:** If it's worth saying, say it. The reader decides importance.
**YClaw example:** "YClaw rewards attention" not "It's important to note that YClaw rewards attention"

### 4. False Depth
**Detect:** "This raises important questions about", "The implications are far-reaching"
**Fix:** State the specific implication. No vague gesturing.
**YClaw example:** "Stakers earn options on floor price" not "This raises important questions about value accrual"

### 5. Performative Nuance
**Detect:** "It's not that simple", "There are many factors to consider", "This is a complex issue"
**Fix:** Name the specific factors. Complexity isn't a conclusion.

### 6. Rule of Three Lists
**Detect:** Three-item lists used reflexively (especially adjective triplets)
**Fix:** Use the number of items the content actually needs. Two is fine. Four is fine.
**YClaw example:** "YClaw makes attention visible and valuable" not "visible, valuable, and verifiable"

## Language Patterns

### 7. Em Dash Overuse
**Detect:** Multiple em dashes (—) in a single paragraph
**Fix:** Use one per piece maximum. Replace extras with periods or commas.

### 8. "Additionally" and Additive Transitions
**Detect:** "Additionally", "Furthermore", "Moreover", "In addition"
**Fix:** Just start the next sentence. The reader follows sequence naturally.

### 9. Copula Avoidance
**Detect:** "serves as", "functions as", "acts as" (instead of "is")
**Fix:** Use "is" when something IS something.
**YClaw example:** "YClaw is an attention rewards protocol" not "YClaw serves as an attention rewards protocol"

### 10. Hollow Verbs
**Detect:** "leverage", "utilize", "facilitate", "implement", "optimize"
**Fix:** "use", "do", "help", "build", "improve"
**YClaw example:** "Stakers use options to borrow USDC" not "Stakers leverage options to facilitate borrowing"

### 11. Hedge Stacking
**Detect:** "It could potentially", "This might possibly", "may perhaps"
**Fix:** One hedge maximum. "It could" or "possibly" — never both.

### 12. Passive Voice (Agentless)
**Detect:** "It has been observed", "It should be noted", "It can be seen that"
**Fix:** Name the actor or delete. "We noticed" or just state the fact.
**YClaw example:** "Creators set their payout split" not "The payout split can be configured"

## Style Patterns

### 13. Uniform Sentence Length
**Detect:** Multiple sentences of similar length in a row (15-25 words each)
**Fix:** Vary rhythm. Mix short punches (3-8 words) with longer explanations.
**YClaw example:** "Watch. Earn. Own the moment." — then follow with a longer sentence.

### 14. Mechanical Paragraphing
**Detect:** Every paragraph is 3-4 sentences. Topic sentence, support, conclusion.
**Fix:** Let paragraph length follow the content. One-sentence paragraphs are fine.

### 15. Colon Introductions
**Detect:** "Here's what you need to know:", "There are several key factors:", "Consider the following:"
**Fix:** Delete the introduction. Start with the first item.

### 16. Symmetrical Structure
**Detect:** Parallel constructions repeated across multiple paragraphs
**Fix:** Break the pattern. Not everything needs the same shape.

### 17. Emotional Bookending
**Detect:** Starting and ending with emotional appeals that bracket factual content
**Fix:** Let facts carry weight. One emotional beat per piece, maximum.

### 18. "Landscape" and "Ecosystem"
**Detect:** "the crypto landscape", "the DeFi ecosystem", "the Web3 space"
**Fix:** Name the specific thing. "Solana DeFi" not "the DeFi landscape"
**YClaw example:** "Solana creator tokens" not "the creator token ecosystem"

## Communication Patterns

### 19. "Delve" and Deep-Dive Language
**Detect:** "delve into", "deep dive", "unpack", "explore"
**Fix:** "Here's how it works" or just explain the thing.

### 20. "Straightforward" Signaling
**Detect:** "The process is straightforward", "Simply put", "In simple terms"
**Fix:** If it's simple, show it. Don't announce simplicity.

### 21. Markdown Formatting in Prose
**Detect:** Bold for emphasis in running text, unnecessary headers, bullet lists where prose works
**Fix:** Let sentence structure do the emphasizing. Save formatting for structured content.

---

## Filler and Hedging

### 22. "Understanding" Padding
**Detect:** "Understanding X is crucial", "To understand this, we need to", "The key to understanding"
**Fix:** Skip to the explanation. The reader is already here.

### 23. Reflexive Hedging
**Detect:** "While there are challenges", "Although not without limitations", "Despite some concerns"
**Fix:** Name the specific challenge or drop it.

### 24. Empty Validation
**Detect:** "This is a valid approach", "There are good reasons for this", "This makes sense"
**Fix:** State why it works, or move on. Don't validate the obvious.

## Brand Voice Conflict Resolution

| Humanizer says | Brand voice says | Resolution |
|---|---|---|
| Use "I" when it fits | "You more than we" | **Brand voice wins** — use "you" |
| Let some mess in | Warm restraint, economical | **Brand voice wins** — precision is the voice |
| Have opinions freely | State facts, not promises | **Brand voice wins** — state mechanics |
| Vary sentence rhythm | Copy breathes, warm restraint | **Aligned** — both want this |
| Be specific, use numbers | Numerals for data | **Aligned** |
| No emojis | No emojis in most channels | **Aligned** |
| Sentence case headings | Sentence case | **Aligned** |

## Quick Self-Check

Before publishing, scan for these high-frequency tells:
1. Does any sentence start with "Additionally" or "Furthermore"?
2. Are there more than one em dash in a paragraph?
3. Do you see "serves as", "functions as", or "acts as"?
4. Is there a sycophantic opener?
5. Are all paragraphs the same length?
6. Any "landscape", "ecosystem", or "delve"?

If yes to two or more, rewrite.
