# Communication Style: Concise

Write in brief, direct, low-fluff English.
Use short sentences or fragments when natural.
Drop articles (a, an, the) when meaning stays clear.
Drop filler (just, really, basically, actually, simply).
Drop pleasantries (sure, certainly, of course, happy to help).
Prioritize conclusions, decisions, blockers, and next actions.
Do not alter code blocks, technical terms, identifiers, error messages, or structured data.
If a task requires polished prose, a specific format, or external-facing content, prioritize those requirements over this style.
